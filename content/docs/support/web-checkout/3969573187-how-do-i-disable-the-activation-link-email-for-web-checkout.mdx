---
title: "How do I disable the activation link email for web checkout?"
---


Context
-------

When using web checkout, customers may receive an automated email containing an activation link from Superwall. For cases, you may want to disable this to control the user experience.

Answer
------

To disable the activation link emails, follow these steps:

1.  Navigate to your Stripe app in the Superwall dashboard
    
2.  Go to Settings / General
    
3.  Locate the "Disable Superwall Emails" setting
    
4.  Change the value from "false" to "true"
    

This setting will affect all automated emails sent by Superwall. Make sure this aligns with your desired user experience before making the change.
