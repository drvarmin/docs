---
title: "How do I find my Superwall Server Secret?"
---


Answer
------

To find your Superwall Server Secret for RevenueCat integration, follow these steps:

1.  Log in to your Superwall dashboard
    
2.  Navigate to Settings
    
3.  Click on Revenue Tracking
    
4.  Select the RevenueCat tab
    
![Superwall Server Secret](/images/superwall-server-secret-rc.png)

The Server Secret is different from your Public Key, which can be found in the general Keys section. Make sure you're specifically in the Revenue Tracking section under Settings to find the Server Secret for RevenueCat integration.
